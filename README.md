文件 1: crawler.py
功能
该文件用于从指定的 API 获取国债数据，并将其保存为 CSV 文件。

主要部分
导入库:
requests: 用于发送 HTTP 请求。
pandas: 用于数据处理和保存为 CSV 文件。
函数定义:
fetch_all_bond_data():
该函数负责发送请求、处理响应和保存数据。
初始化请求参数（如 page_no 和 page_size）。
使用一个循环不断请求数据，直到获取所有债券信息。
每次请求时，提取所需的债券信息并存储在列表中。
最后，将所有数据保存为 treasury_bonds_2023.csv 文件。
请求和响应处理:
使用 POST 方法向指定的 URL 发送请求，并处理返回的 JSON 数据。
检查响应状态，确保请求成功。
数据提取:
提取每个债券的关键信息（如 ISIN、债券代码、发行人、债券类型、发行日期和最新评级）。
保存数据:
使用 pandas 将提取的数据保存为 CSV 文件，以便后续分析和使用。
使用方法
运行该文件将自动从 API 获取国债数据并生成 CSV 文件。

文件 2: reg_search.py
功能
该文件实现了一个自定义的正则匹配函数，用于从文本中提取特定信息，包括股票代码和换股期限日期。

主要部分
导入库:
re: 用于正则表达式匹配。
json: 用于处理 JSON 数据格式。
datetime: 用于日期处理。
辅助函数:
is_date_format(value): 检查字符串是否符合日期格式。
format_date(value): 将日期字符串格式化为 YYYY-MM-DD。
主函数:
reg_search(text, regex_list):
该函数接受文本和正则表达式列表，返回匹配的结果。
遍历每个正则表达式，使用 re.findall() 获取匹配项，并支持跨行匹配。
对于匹配的日期进行格式化处理。
测试示例:
包含一个示例文本和相应的正则表达式列表。
调用 reg_search() 函数并打印结果，以便验证功能。
使用方法
运行该文件将测试正则匹配功能，并输出匹配结果的 JSON 格式。
